# Getting Started with Python

Welcome to your Python journey! This guide will help you set up Python and write your first program.

## Installing Python

### Windows

1. Download from [python.org](https://python.org)
2. Run the installer
3. Check "Add Python to PATH"
4. Click "Install Now"

### macOS

```bash
brew install python
```

### Linux

```bash
sudo apt-get update
sudo apt-get install python3
```

## Your First Program

```python
print("Hello, World!")
```

## Using the REPL

Python's interactive shell is perfect for experimentation:

```python
>>> 2 + 2
4
>>> name = "Python"
>>> print(f"I love {name}!")
I love Python!
```

## Running Python Files

Save your code in a `.py` file:

```bash
python hello.py
```

## Conclusion

You're now ready to start your Python journey!
